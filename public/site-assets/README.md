# Website Assets Folder

Put all your website assets here.

## Structure
- `public/site-assets/images` -> photos, hero images, section images
- `public/site-assets/logos` -> brand and media logos
- `public/site-assets/icons` -> icons/badges
- `public/site-assets/docs` -> PDFs or downloadable files

## How to use in React
Files in `public/` are available directly by URL.

Examples:
- `public/site-assets/images/hero-dr-ishinna.png` -> `/site-assets/images/hero-dr-ishinna.png`
- `public/site-assets/logos/tedx.png` -> `/site-assets/logos/tedx.png`
